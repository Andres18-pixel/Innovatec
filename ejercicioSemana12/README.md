# Innovatec – Árbol y Grafo

**Proyecto Final – Estructuras de Datos**

## Descripción
Programa en C# que cumple con el caso de estudio “Parque Tecnológico Innovatec”:

- **Parte A**: Árbol general para la jerarquía organizativa  
  Funcionalidades: inserción, recorrido preorden, búsqueda, conteo y cálculo de niveles.

- **Parte B**: Grafo no dirigido y ponderado para rutas entre edificios  
  Funcionalidades: registro de edificios y conexiones, visualización, validación de conexidad (DFS) y ruta más corta
